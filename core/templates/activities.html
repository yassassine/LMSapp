L'inscription aux activités se fait directement depuis votre espace parent :

  1. Connectez-vous à votre compte
  2. Accédez à la section "Activités"
  3. Sélectionnez l'activité souhaitée
  4. Validez l'inscription

Pour les activités spéciales, une réservation préalable est nécessaire via la page de l'activité.

La plupart de nos activités régulières sont incluses dans les frais de garde de base. Cela comprend :

  • Les activités artistiques quotidiennes
  • Les jeux éducatifs
  • Les séances de motricité
  • Les ateliers découverte

Certaines activités spécialisées (robotique, musique avancée, sorties spécifiques) peuvent faire l'objet d'un supplément. Ces informations sont toujours précisées sur la fiche activité.

Nous comprenons que les enfants peuvent changer d'intérêt. Voici notre politique :

  • Pour les activités hebdomadaires : possibilité de changer après un mois
  • Pour les activités ponctuelles : remboursement possible jusqu'à 48h avant
  • Nous proposons toujours une séance d'essai pour les nouvelles activités

N'hésitez pas à discuter avec nos éducateurs pour trouver une activité mieux adaptée à votre enfant.

Tous nos animateurs sont rigoureusement sélectionnés selon ces critères :

  • Diplôme dans le domaine d'activité ou expérience significative
  • Certificat de qualification en animation
  • Casier judiciaire vierge
  • Formation aux premiers secours pédiatriques
  • Test pédagogique et entretien avec notre équipe

Nous renouvelons régulièrement nos formations pour garantir la qualité de nos activités.
